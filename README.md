# debian-wincent

!! Work in progress !!

## Install
### Dependencies
    git
    make

### Instructions
    git clone https://github.com/Jonawi1/debian-wincent.git
    cd debian-wincent
    sudo make
    sudo reboot

If you for any reason need to do a reinstall run:
    
    sudo make clean install
